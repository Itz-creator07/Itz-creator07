## 💻 Projects

### X-IDS (Explainable Intrusion Detection System)

Research project focused on solving alert fatigue in security operations. Making AI decisions transparent so analysts can act with confidence instead of ignoring alerts or wasting time investigating blind.

**Approach:**
- Interpretable ML methods that expose feature contribution
- Multi-signal detection mimicking analyst reasoning
- Transparent combination of diverse indicators
- Design priority: actionable clarity over marginal accuracy gains
